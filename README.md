## 📝 Описание продукта
Тестируемая страница — конструктор виджета **«Календарь мероприятий»** от 3S.INFO (iGaming-портал). Виджет позволяет настроить параметры (тематика, страны, размер блока, цветовая тема) и получить готовый `iframe`-код для встраивания на сторонний сайт. Доступен на двух языках: русском (`/eventswidget/`) и английском (`/en/eventswidget/`).

## 🐛 Баг-репорты

### 🔴 Critical
<details>
<summary><b>BUG-003: Тема «Красная» — параметр theme отсутствует в коде iframe</b></summary>

* **URL:** `https://3s.info`
* **Шаги воспроизведения:**
  1. Открыть страницу.
  2. Шаг 4 → выбрать красную тему (светлая, третий вариант).
  3. Посмотреть код в поле внизу страницы.
* **Ожидаемый результат:** `src="...?theme=red&event_lang=ru"`
* **Фактический результат:** `src="...?event_lang=ru"` — параметр `theme` полностью отсутствует.
* **Влияние:** Пользователь получит дефолтную (синюю) тему вместо красной.
</details>

### 🟠 Major
<details>
<summary><b>BUG-001: На английской версии вверху страницы отображается пустой блок</b></summary>

* **URL:** `https://3s.info`
* **Шаги воспроизведения:**
  1. Открыть русскую версию, переключить язык на English.
  2. Открыть DevTools → Network.
* **Ожидаемый результат:** Страница отображается корректно.
* **Фактический результат:** Большой пустой блок в верхней части. GIF-файлы (`6.gif` и `7.gif`) возвращают **404**.
</details>

<details>
<summary><b>BUG-002: Mixed Content — карусель промокодов пустая</b></summary>

* **URL:** `https://3s.info` (обе версии)
* **Шаги воспроизведения:** Прокрутить до секции промокодов в футере, открыть Console.
* **Ожидаемый результат:** Все карточки отображают картинки.
* **Фактический результат:** 3 из 4 карточек пустые (чёрные). В консоли 10+ предупреждений *Mixed Content* (ссылки в `footer-promocodes-slider.js` прописаны по HTTP).
</details>

### 🟡 Minor
<details>
<summary><b>BUG-004: Лейбл «Предыдущий пост» наезжает на карточку в «Других материалах»</b></summary>

* **URL:** Страница отдельного события.
* **Фактический результат:** Текст и дата визуально перекрывают карточку. CSS-баг (`overflow`/`z-index`).
</details>

<details>
<summary><b>BUG-005: Import map конфликты JS-модулей в консоли (RU-версия)</b></summary>

* **Фактический результат:** 10+ предупреждений `"An import map rule for 'app' was removed..."`. Затронуты модули Symfony/Stimulus.
</details>
